# espWeather
Solar powered Wemos D1 Pro with BME280

## BOM
* Wemos D1 Pro
* BME280
* 2x Solar Cell (6V 1W 60x110mm)
* MCP1700-3302E (3.3V)
* 120k Resistor
* 1000uF Electrolytic Capacitor
* 2x 100nF Ceramic Capacitor
* TP4056 (Li-Ion charger)
* 18650 Li-Ion Battery
* 18650 Battery Holder
* 80x44mm Proto PCB
